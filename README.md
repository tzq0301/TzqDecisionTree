# TzqDecisionTree
《机器学习（周志华）》习题4.3

## main.ipynb
`main.ipynb`是项目代码的入口脚本。

## DecisionTree实验报告.docx
该文件是描述本实验的实验报告。

## data
data文件夹中包含：
- **watermelon.csv**：《机器学习（周志华）》表4.3的数据
- **data.py**：相关的数据处理脚本

## model
model文件夹中的**tree.py**包含了四个类：
- **DecisionTree**：决策树
- **DecisionTreeNode**：决策树节点
- **DecisionTreeTrainer**：决策树训练器
- **Accuracy**：度量精度的封装类

## utils
utils文件夹中封装了模型构建、训练等环节中的一些工具函数。
